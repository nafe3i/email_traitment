Ce dossier doit contenir :
  - credentials.json  (téléchargé depuis Google Cloud Console)
  - token.json        (généré automatiquement au premier lancement)

⚠️  NE JAMAIS committer ces fichiers sur Git (déjà dans .gitignore)

Instructions :
1. Créez un projet sur https://console.cloud.google.com
2. Activez l'API Gmail
3. Créez des identifiants OAuth2 de type "Desktop Application"
4. Téléchargez le fichier JSON → renommez-le credentials.json
5. Placez-le dans ce dossier
6. Lancez main.py → il ouvrira un navigateur pour autoriser l'accès
7. token.json sera créé automatiquement
